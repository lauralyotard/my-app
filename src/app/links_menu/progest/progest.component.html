Projet de conversion Access en Web

L'objectif

L’objectif principal du projet était de transférer une interface accessible via Access en version Web disponible depuis le net. Le site a pour but d'améliorer l'usage actuel des utilisateurs. Il était essentiel de rendre le site plus intuitif, plus fluide et plus léger. Les pages Access servaient pour la consultation et saisie de la comptabilité de la Fédération Française de Triathlon.

Le site est développé avec le Framework Symfony sur l'architecture logicielle Modèle-Vue-Contrôleur.

Le plus grand défi a été de connecter et de travailler avec les données existantes en base. Celle ci se trouvait sur une base Microsoft SQL Server. Il y avait énormément de datas, il fallait faire des vues afin de faire appel à des requêtes simples et minimiser le temps de chargement des données appelées.

Le second point principal du projet était de segmenter les autorisations d'accès avec des pages de connexion et restrictions d'accès par hauteur de rôles. Il fallait sécuriser l'ensemble des pages pour supprimer tous risques qu'un utilisateur se retrouve sur une page où l'accès lui est restreint.

Un dernier point essentiel était de composer la charte graphique du site. Etant donné, qu'il émane d'une base Access, il n'avait aucune réelle identité graphique. Le site concerne Fédération Française de Triathlon, il fallait utiliser les mêmes couleurs que le site officiel.
